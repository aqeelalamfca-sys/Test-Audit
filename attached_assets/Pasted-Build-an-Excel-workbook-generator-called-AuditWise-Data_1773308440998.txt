Treat GL Code as the unique matching key unless a module specifically requires document-level matching. The workbook must work for these 5 source sheets:

1. Trial Balance
2. General Ledger
3. Bank
4. Accounts Receivable
5. Accounts Payable

Create exactly 5 formatted validation sheets in one workbook, with strong headings, color coding, filters, frozen panes, data validation dropdowns where useful, conditional formatting, formulas, exception flags, auto totals, and review remarks columns. The workbook must be usable even when source data is pasted manually by end users. Keep the layout simple, practical, and audit-friendly.
